Introduction

rgbmultiselect is a jQuery plugin that turns standard HTML SELECT fields (single and multiple) into easier-to-use widgets. Users can more easily select and filter options and use both the mouse and keyboard to interact with the options. It has a variety of features and configuration options, including:
  • widget fits in the space of a single text input field
  • supports sticky and exclusive options
  • can set a maximum limit on the number of options selected
  • all selections stored using the backing SELECT box for maximum compatibility and degradability
  • cross-browser: IE 6+, Firefox 2+, Safari 3+, Opera 9+, Chrome 1+
  • small size (24K minified, 14K packed)

Setup and Configuration

Select list setup:

rgbmultiselect builds its option list off of an existing SELECT list; options will be listed in the order in which they appear in the SELECT list. You can set the following flags in the "rel" attribute of each <OPTION> to alter the way options are displayed. Note that you can set more than one option, separated by spaces (<OPTION value="1" rel="sticky exclusive">One</OPTION>):
  • default: this is a default option and should be omitted from the list altogether.
  • sticky: this option should always appear at the top of the list (immediately below the Clear All / Select None option, if available; see below) and should not move if unselected.
  • exclusive: if this option is selected, all other options will be unselected. Additionally, if an exclusive option is already selected and another option is then selected, the exclusive option will be unselected.
Note that, if you wish to make all options exclusive, use the allOptionsExclusive configuration option instead. This configuration option will be forced if the SELECT list does not have the "multiple" attribute set.

Specifying configuration options:

Configuration options are specified as follows (first example has no configuration options, second example has two options): 
  $("#picknumbers").rgbmultiselect();

  $("#pick3numbers").rgbmultiselect({
				      maxSelections: 3,
                                      helpText: "Select some options"
				    });

fieldTextFormatOnBlur examples:

The fieldTextFormatOnBlur options are designed to give you a good amount of flexibility around how selected options will be displayed when the user enters the field. Let's look at a few examples:
  • User selects "One", "Two", "Three", "Four", "Five"
    • fieldTextFormatOnBlur = "%o"
    • fieldTextFormatOnBlurNumToShow = -1
    • fieldTextFormatOnBlurIfLTENumToShow = "%o"
    • On blur, input field will display "One, Two, Three, Four, Five"
    • fieldTextFormatOnBlur = "%c: %o"
    • fieldTextFormatOnBlurNumToShow = -1
    • fieldTextFormatOnBlurIfLTENumToShow = "%c: %o"
    • On blur, input field will display "5: One, Two, Three, Four, Five"
    • fieldTextFormatOnBlur = "%c: %o + %a more"
    • fieldTextFormatOnBlurNumToShow = 3
    • fieldTextFormatOnBlurIfLTENumToShow = "%c: %o"
    • On blur, input field will display "5: One, Two, Three + 2 more"
    • fieldTextFormatOnBlur = "%c: %o + %a more"
    • fieldTextFormatOnBlurNumToShow = 5
    • fieldTextFormatOnBlurIfLTENumToShow = "%c: %o"
    • On blur, input field will display "5: One, Two, Three, Four, Five"

Back button behavior:

Some browsers don't support auto-filling some form fields when the user clicks "Back" if form fields have been added using JavaScript. To address this issue, if you specify an input field with an ID in the pattern expected by the plugin, it will detect and use that input field instead of creating one.

For example, if you have a SELECT box with an ID of "auto_manufacturers", the plugin will first look for an input field with the ID "auto_manufacturers_rgbmultiselect" before creating an input field on its own.

Additionally, if you want your input field to be a different size than your SELECT box, you can set that size as a style on the input field and the plugin won't alter it. If the plugin creates the input field, it will make it have the same width as the original SELECT box.
